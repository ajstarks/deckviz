With decksh you can make many kinds of lists:
plain, bulleted, numbered, centered.
\n
The type of list is specified by the keyword: 'list' for plain,
'blist' for bulleted, 'nlist' for numbered, 'clist' for centered.
\n
The arguments for lists include the x and y location and the text size of the list.
\n
For example to begin a list at (20, 40) with text size 3 use:
\n
list 20 40 3
\n
Following the list type is any number of list items (li "item name"),
one per line. The list ends with "elist" on a line by itself.
\n
Decksh does not have native support for tables, but you can construct tables using "parallel lists", by aligning a series of plain lists to a set of columns.
