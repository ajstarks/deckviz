Variables provide a method for attaching a name to an object
so that the name may be used for other operations.
\n
Variables may refer to either numbers, or text strings (data defined between " characters).
Use the '=' after the name to assign the value of variable.  The general syntax is:
\n
name = value
\n
In the example, (1) the value of "hello" is assigned to s, (2) 5 and 50 to the variables x and y and (3) ts is assigned the value of 5. Next, use the variables: 
\n
text s x y ts
\n
which is equivalent to:
\n
text "hello" 5 50 5
\n
When using variables that contain numbers, you can also perform binary operations:
add (+), 
subtract (-), 
multiply (*),
divide (/), and modulo (%). For example,
\n
a = x + y  // a gets the sum of x and y
\n
You can also use 'assignment operations' on a single variable, for example to add 10 to an existing variable x, use:
\n
x += 10
